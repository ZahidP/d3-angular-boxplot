This readme could probably use some work, let me know if I can improve.

Controller:
'$scope.selection' array: the buttons underneath the boxplot are used to push new indices
which you want to graph into this $scope.selection. It is then passed into the directive.

(boxPlot) Directive:
scope.render(data): 'scope.selection' is passed as the data object. This will select new objects to graph. Additionally, we will watch for changes in the data object, and update the chart when it has changed.

Changes to original box.js from bl.ocks.org:
D3 integration, can add and remove boxes. The data, however, is actually passed in through the directive and not the controller..This can be changed if needed.

This is used to remove any extra boxes. See: http://bost.ocks.org/mike/circles/ for explanation on enter() and exit() .

Essentially, enter() is used whenever an additionaly element enters your dataset, and exit() is used whenever an element is removed.
